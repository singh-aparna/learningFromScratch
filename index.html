  74. What is API call? (asked in interview)

  75. Definition and key difference of Promise.all, Promise.allSetteled, Promise.race and Promise.any.

    Promise.all ‚Üí Resolves when all promises succeed, rejects immediately if any fail.
    Promise.allSettled ‚Üí Waits for all promises to settle (either resolve or reject), always returns an array of results.
    Promise.race ‚Üí Resolves or rejects as soon as the first promise settles, ignoring others.
    Promise.any ‚Üí Resolves when any one promise succeeds, rejects only if all fail.
